Class: Orders

Orders()

A Orders collection is a collection of Order models that represent one or more DataONE service Orders, as part of the DataONE Bookkeeper service. See https://github.com/DataONEorg/bookkeeper for documentation on the DataONE Bookkeeper service and data model.
