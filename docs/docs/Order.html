Class: Order

Order()

A Order Model represents a single instance of a Order object from the DataONE Bookkeeper data model. Orders represent a Product that has been ordered by a Customer and is paid for on a recurring basis or is in a free trial period. See https://github.com/DataONEorg/bookkeeper for documentation on the DataONE Bookkeeper service and data model.
